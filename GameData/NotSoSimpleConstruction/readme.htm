Not So SimpleConstruction! (NSSC)

A SimpleConstruction! expansion which requires SimpleConstruction! and Kerbal Space Program.

By zer0Kerbal from MrCarrot, originally by RealGecko

adopted with express permission and brought to you by KerbSimpleCo

Features

  • Parts Added
    • Mallet
    • Survey Stake
    • Recycler
      • Small
      • Large
    • Disposable Pad
    • Orbital Dock
  • Features Added
    • Adds Survey Station to the Cupola.
    • Optional patch included to add Survey Station to all crewed command parts (read file for instructions).
