# Not So SimpleConstruction

A [SimpleConstruction!][scon] expansion which requires [SimpleConstruction!][scon] and Kerbal Space Program.

## By [`zer0Kerbal`][zer0Kerbal] from [`MrCarrot'][mrcarrot], originally by [`RealGecko`][realgecko]

### Features

* Parts Added
  * Mallet
  * Survey Stake
  * Recycler
    * Small
    * Large
  * Disposable Pad
  * Orbital Dock
  * Control Reference
* Features Added
  * Adds Survey Station to the  Cupola.
  * Optional patch included to add Survey Station to all crewed command parts (read file for instructions).
